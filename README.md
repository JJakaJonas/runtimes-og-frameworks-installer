# runtimes-og-frameworks-installer
En samling scripts der installere forskellige runtimes og frameworks samt danske sprog pakker hvis de eksistere til softwaren.

<H2>Pakken indeholder lige nu følgene software.</H2>

<H3>Microsoft Visual C++ (x86/x64)</H3>
2005 (VC++ 8.0)
<br>
2008 (VC++ 9.0)
<br>
2010 (VC++ 10.0)
<br>
2012 (VC++ 11.0)
<br>
2013 (VC++ 12.0)
<br>
2015-2026 (VC++ 14.0)

<H3>Microsoft .NET Framework (x86/x64)</H3>
3.5 SP1
<br>
4.8.1
<br>
4.8.1 Dansk sprog pakke

<H3>Andet</H3>
DirectX 9c
<br>
OpenAL
<br>

<br>
Flere kan komme med tiden

<H2>Sådan bruges disse scripts</H2>
For at installere softwaren skal du blot åbne install_all.bat filen i rod mappen og sige ja til at køre det som administrator. 
<br>
Så klare scriptet resten.
